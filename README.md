# 🐍 Python Challenge Hub (En Construcción 🏗️)

¡Bienvenido a **Python Challenge Hub**! Este repositorio es una colección de ejercicios de Python diseñados para ayudarte a mejorar tus habilidades de programación, desde los conceptos básicos hasta desafíos más complejos.

> [!NOTA]
> Este repositorio se encuentra actualmente **en desarrollo**. Estoy añadiendo nuevos ejercicios y soluciones semanalmente.

---

## 🚀 Estructura del Repositorio

Los ejercicios están organizados por niveles de dificultad para que puedas avanzar a tu propio ritmo:

| Nivel | Descripción | Conceptos Clave |
| :--- | :--- | :--- |
| **Básico** 🟢 | Fundamentos básicos de Python. | Variables, tipos de datos, bucles `for/while`, condicionales. |
| **Intermedio** 🟡 | Aplicación de lógica y estructuras. | Listas comprimidas, diccionarios, funciones, manejo de errores. |
| **Avanzado** 🔴 | Desafíos de algoritmos y optimización. | Decoradores, generadores, POO avanzada, algoritmos de búsqueda. |

---

## 📁 Cómo usar este repositorio

Cada carpeta de nivel contiene subcarpetas con el nombre del ejercicio. Dentro de cada una encontrarás:

1. `ejercicio.py`: El archivo donde puedes escribir tu código.
2. `solucion.py`: Mi propuesta de solución comentada.

### Ejemplo de flujo de trabajo:
1. Navega a la carpeta del nivel que te interese.
2. Intenta resolverlo por tu cuenta en `ejercicio.py`.
3. Compara tu resultado con la `solucion.py`.

---
